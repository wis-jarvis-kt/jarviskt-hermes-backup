# Brain-First Lookup Protocol (5 Steps)

Before using ANY external API to research a person, company, or topic:

1. `search "name"` — keyword search for existing pages
2. `query "natural question about name"` — hybrid search for context
3. `get <slug>` — if you know the slug, read the full page
4. Check backlinks: who references this entity?
5. Check timeline: recent events involving this entity

The brain almost always has something. External APIs fill gaps, not start from scratch.

## When to use

- Before any external research on a person/company
- Before answering questions about entities you might have context on
- On every inbound signal that mentions an entity

## The Rule

Always brain-first. The brain is your single source of truth for everything
you already know. External APIs only fill gaps.
