## HLA Life Insurance Premium Statement Workflow (learned 2026-03-22)

**Full flow discovered by watching Master KT:**
1. Login to HLA portal (ktgoo2880 / PortalKT88-2)
2. Navigate to: `https://portal.hla.com.my/cms/Servicing/Policy-Info.aspx` (session context)
3. Then navigate to: `https://portal.hla.com.my/LifeInsurancePremium/WebUI/main.aspx?GLLLLYR1B0ALRYRYGLO10BX0X9LBAXORR0O9`
4. Fill form:
   - Agent code dropdown: `cphContent_ddlSearchAgentCode` (A0022418 or A0037263)
   - Life Assured Name: `cphContent_txtLifeAssuredName`
   - Year: `cphContent_cboDUYear` (2025, 2024, 2023)
5. Click Get Record: `cphContent_btnGetRec`
6. Find ViewPdf links — index 5 = row 6 (0-based): `cphContent_dgLifeInsurancePremiumAG_Hyperlink1_5`
7. Navigate to the PDF URL directly (session cookies carry over)
8. Use CDP `Page.printToPDF` to download
